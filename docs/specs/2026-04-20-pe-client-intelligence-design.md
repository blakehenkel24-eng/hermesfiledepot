## Goal

Build a private, team-shared PE client intelligence tool that surfaces a curated morning brief of news and opportunities across the PE firms the Inspire11 PE services team tracks. The tool combines automated news aggregation with AI synthesis and pre-researched firm profiles.

## Users

- The full PE services team at Inspire11 (5-10 people)
- No authentication -- anyone with the link can access
- All team members see the same data, same notes

## Aesthetic Direction

**Editorial Private Bank** -- Financial Times meets private wealth management morning brief.

- Light mode only. Cream/warm white background (#FAFAF8)
- Warm serif display font (Georgia) for headings
- Clean sans-serif body (system font stack)
- Gold/amber accent color (#B8860B) for opportunities, alerts, and interactive elements
- No gradients, no glassmorphism, no neon, no rounded cards everywhere
- Rigid grid with visible structural dividers
- Feels like opening a morning intelligence packet at a white-shoe firm
- Content-first: maximum readability, minimum decoration

## Tracked Firms

- Initial scope: 25-50 US PE firms
- Heavy Chicago emphasis but national coverage
- Firms researched and pre-populated by the builder (Hermes), not manually entered by the user
- User can add additional firms later through an "Add Firm" form

## Data Sources (All Free)

1. **Google News RSS** -- filtered by firm names and PE industry terms
2. **GDELT API v2** -- event detection for acquisitions, fundraises, executive moves
3. **PE Hub free articles** -- PE-specific M&A and fundraising coverage
4. **Axios Pro Rata newsletter** -- daily deal/fundraising digest (parsed)
5. **Firm websites / press releases** -- official announcements
6. **SEC EDGAR** -- Form D filings for fundraising, Form ADV for firm changes
7. **LinkedIn public pages** -- personnel moves, company updates
8. **OpenAI GPT-4o-mini** -- AI synthesis of news into morning brief summaries

## Architecture

### Three Main Views

**1. Dashboard (Home -- `/`)**

The morning intelligence brief. This is what opens first.

- **Top section: AI Morning Summary** -- A 3-5 paragraph synthesis of the most important developments across all followed firms in the last 24 hours. Generated by OpenAI from the aggregated news. Each paragraph covers a theme: "Fund Activity", "Personnel Moves", "Portfolio Highlights", "Actionable Opportunities". Rendered in warm serif. Gold horizontal rule above it. Feels like reading a partner's morning note.

- **Middle section: News Feed** -- Chronological list of all articles/events detected. Each item shows:
  - Source icon (Google News, GDELT, SEC, etc.)
  - Firm name (clickable to firm profile)
  - Category tag: Acquisition | Fund | Portco | Personnel | Strategy | Opportunity
  - Headline (linked to original article)
  - 1-2 sentence AI-generated summary
  - Timestamp
  - Optional "Opportunity" badge (gold) if AI detects a consulting trigger

- **Right sidebar: Signal Heat Index** -- List of followed firms sorted by activity volume in the last 7 days. Firms with the most news float to the top. Shows: firm name, tier badge (1/2/3), news count this week, latest headline snippet. Acts as a quick-glance triage tool.

**2. Firm Profile (`/firm/[slug]`)**

Deep dive into a single PE firm.

- **Header block:** Firm name (large serif), HQ city/state, website link, tier badge, date added
- **Overview section:** Firm description (2-3 sentences), AUM (if available), fund count, investment strategy/focus areas (listed as tags), industry verticals, geographic focus
- **Key Personnel table:** Name, Title, Office location, LinkedIn link (if public). Partners, MDs, Operating Partners, Principals. Sorted by seniority.
- **Portfolio Companies:** Company name, industry, investment date (if known), status (active / exited). Simple table.
- **Recent News:** Same news feed as dashboard but filtered to this firm only
- **Team Notes:** Shared notepad at the bottom. Any team member can add notes ("Spoke with John Smith about AI DD capability -- follow up Q3"). Stored in a JSON file on the server. Simple textarea, save button, timestamp + author field (free text, no auth).

**3. Client Tracker (`/tracker`)**

Manage the list of followed firms.

- **Firm list:** All followed firms in a table. Columns: Firm Name, Tier, HQ, Focus, Last News Date, # Articles This Week, Actions (Edit / Remove)
- **Tier system:** Tier 1 (active client, gold badge), Tier 2 (warm relationship, silver), Tier 3 (prospecting, bronze). Tiers affect how prominently firms appear in the dashboard.
- **Add Firm form:** Name, website, tier, HQ city. On submit, triggers a research pipeline that auto-populates the profile within minutes.
- **Bulk import:** Paste a list of firm names, system researches each one.

### Data Layer

- **Firm profiles:** Pre-researched JSON files in `/data/firms/`. One file per firm containing all profile data (description, team, portfolio, strategy). Built upfront by Hermes.
- **News articles:** Stored in `/data/news-cache.json`. Updated by cron jobs. Rolling 30-day window.
- **Team notes:** Stored in `/data/notes.json`. Simple key-value store keyed by firm slug.
- **AI summaries:** Cached in `/data/summaries/`. One per day. Regenerated each morning.
- **No database.** File-based storage for v1. Simple, deployable, zero config.

### API Routes

- `GET /api/news` -- Returns cached news articles, accepts query params for firm filter, category filter, date range
- `GET /api/firms` -- Returns all followed firms with profile data
- `GET /api/firms/[slug]` -- Returns single firm profile with recent news
- `POST /api/firms` -- Add new firm (triggers research pipeline)
- `PATCH /api/firms/[slug]` -- Update firm tier or basic info
- `DELETE /api/firms/[slug]` -- Remove firm from tracker
- `GET /api/summary` -- Returns today's AI morning summary (cached)
- `POST /api/notes/[slug]` -- Save a team note for a firm
- `GET /api/cron/fetch-news` -- Cron endpoint (every 4 hours). Fetches RSS + GDELT for all followed firms, stores new articles, tags categories
- `GET /api/cron/generate-summary` -- Cron endpoint (daily at 6am CT). Generates AI morning summary from last 24h of news

### Cron Schedule

- News fetch: Every 4 hours (`0 */4 * * *`)
- AI summary generation: Daily at 6:00 AM Central (`0 11 * * *` -- UTC)
- Both configured in `vercel.json`

## Pre-Populated Data

Hermes will research and build complete profiles for 25-50 US PE firms before launch. Each profile includes:

- Firm name and slug
- Description (2-3 sentences)
- HQ city and state
- Website URL
- AUM (latest available)
- Number of funds
- Investment strategy and focus areas
- Industry verticals
- Key personnel (name, title, office)
- Notable portfolio companies (name, industry, status)
- Initial tier assignment (suggested, user can change)

The user provides the initial list of firm names. Hermes does all the research.

## Shareable Links

- Any page can be shared via URL
- No auth, no accounts
- The morning summary gets a permalink: `/summary/2026-04-20`
- Individual firm profiles are always linkable: `/firm/thistledown-capital`

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS with custom design tokens
- **Fonts:** Georgia (display) + system sans-serif stack (body)
- **AI:** OpenAI GPT-4o-mini via AI SDK
- **RSS:** rss-parser library
- **Events:** GDELT API v2 (free, no key)
- **Deployment:** Vercel
- **Storage:** File-based JSON (no database)
- **Language:** TypeScript

## Out of Scope (v1)

- Email digests (can add later)
- Individual user accounts / auth
- Push notifications
- Mobile app
- Integration with CRM tools (DealCloud, Salesforce)
- Historical data beyond 30 days
- Automated LinkedIn scraping (manual research only)
- Real-time WebSocket updates

## Success Criteria

- Team can open the tool each morning and get a 60-second overview of what happened across all their PE clients
- Clicking into a firm gives a complete profile with team, strategy, portfolio, and recent news
- AI surfaces at least one actionable opportunity per week that the team wouldn't have caught otherwise
- Adding a new firm takes under 5 minutes from name to full profile
- The interface feels like a premium morning brief, not a SaaS dashboard
